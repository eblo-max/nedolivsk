# Инструменты баланса боёвки

- `sim_matrix.py` — матрица винрейтов: 6 эталонных билдов × весь бестиарий.
  Запускать до/после ЛЮБОГО ребаланса (статов, врагов, формул, черт).
- `tune_enemies.py` — авто-тюнер: подбирает hp/attack зверя под целевой
  винрейт «билда-владельца» (цели в TARGETS). Черты меняют силу зверя —
  после новой черты прогонять заново.

Полосы сторожит CI: `tests/test_balance_matrix.py` (упал — цифры из тюнера
вбить в `bot/game/combat.py` и обновить коридоры осознанно).

Запуск: `python tools/balance/sim_matrix.py` (BOT_TOKEN подставится сам).
