# 🏛 Общие стройки — «Чудеса города» + валюта «Зодар»

Кооперативная мегастройка по модели **FFXIV Ishgard Restoration** + двухслойные
награды **Elite Dangerous** + карта как холст достижения. Каждый игрок строит
СВОЮ таверну, но **чудеса строит весь город вместе**, и это подпись игры —
единый живой город (см. память `world-unified-global-city`).

## Зафиксированные решения
- **Валюта: «Зодар»** ⚒ (`Player.zodar`). Лор: Артель зодчих переплавляет
  инструмент умершего мастера в слиток и чеканит зодары — их вручают за труд,
  достойный лечь рядом с трудом мастеров.
- **Три закона зодара (ради здоровья экономики — урок FFXIV):**
  1. не купить за золото; 2. не продать/передать (**bind-on-earn**);
  3. только за участие в стройке. Нарушь любой — это «золото с шагами».
- **Награды:** **престиж + эксклюзивные рецепты** на новую еду/питьё (мягкий
  эдж заработан участием, не деньгами; кормит цель «мотивировать варить»).
- **Первое чудо: «Твердыня»** (стена/детинец) — асет-безопасно (стены есть в
  любом паке), фазы читаются «растёт сегмент за сегментом», бафф felt (легче
  отбить Орду), вшивается в живую механику `invasion`.

## Модель
- **Одно живое чудо на весь мир** (таблица `wonders`, паттерн `RaidBoss`/`Invasion`).
- **Фазы** (Твердыня: Ров → Стены → Детинец). Вклад = ресурсы/**готовые блюда**/
  золото → очки **по ценности** (`RESOURCE_PRICE` / цены `GOODS`). Цель фазы
  масштабируется по числу активных (как `market_scale`).
- **Убывающая отдача** на спам дешёвкой + дневной порог (анти-кит; урок Ан'Киража).
- **Зодары** начисляются вкладчикам: по ходу + **перцентильный бонус** на
  закрытии фазы (топ-10/25/50/75%). Не строил — зодаров нет (глоб-бафф всё равно всем).
- **Капстоун:** кто заложил последний камень фазы — именной трофей + имя навечно
  в летопись/на монумент.
- **Глобальный бафф** чуда включается на финише (флаг в `world.live`, читается
  котировками по правилу **показ = действие**, как `fgoal.feast_mult`).

## Лавка Артели (сток зодаров — Фаза 2)
Престиж (титулы, скин таверны, имя в монументе), **эксклюзив-рецепты** (уникальные
блюда/напитки только зодчих → живой P2P-спрос), мягкий QoL (доп. слот биржи/
вылазки, ускорение варки), городское (право выдвинуть чудо). Цены 10–150 ⚒;
топ-престиж = «несколько чудес» (долгий чейз). Ротация наград по сезонам.

## Экономическая целостность
- Зодар — **закрытый контур**: не инфлирует золото, кит не купит престиж.
- Вклад — **крупный новый сток** золота/ресурсов (категория `wonder` в `economy.py`).
- Награды — престиж/косметика/QoL, **не пауэр-крип** (эдж максимум через эксклюзив-рецепты).

## План по фазам
- **Фаза 0 — фундамент данных (СДЕЛАНО):** `Player.zodar`, таблица `wonders`,
  сток `wonder` в econ, реестр `bot/game/wonder.py`, миграция, этот док.
- **Фаза 1 — ядро цикла (сервер):** вклад→очки→зодар (курс, убыв. отдача,
  дневной порог), фазы, перцентильные бонусы, капстоун, `active_bonuses`,
  вшить бафф Твердыни в `invasion`; `/api/wonder` + `/api/wonder/contribute`;
  анонс/летопись в нотифаере; тесты (вес очков, анти-дюп, показ=действие).
- **Фаза 2 — Лавка Артели:** каталог, покупка, применение (титул/скин/рецепт/
  слот); инвариант «не торгуется»; вшить эксклюзив-рецепты в производство.
- **Фаза 3 — мини-апп + карта:** экран «Стройка города» (полоса, фазы, вклад,
  доска вкладчиков, баланс зодаров), Лавка; рендер чуда на `/world` по фазам
  (плейсхолдер → финальный спрайт стены + оверлей лесов); гейты build/превью.
- **Фаза 4 — выбор чуда + сезонность:** ротация/голосование, сезонные награды.
- **Фаза 5 — гейты, анти-абуз, прод-проба, анонс.**

## ⚠ Замечание по деплою (важно)
`create_tables()` через `create_all` таблицы только СОЗДАЁТ, но существующие НЕ
меняет. Значит: пока схема `Wonder` не финальна, **не деплоить Фазу 0 в отрыве от
Фазы 1** — иначе таблица `wonders` создастся с ранней схемой, а колонки, добавленные
в модель позже, на проде не появятся. Правило: либо катим Фазы 0+1 вместе (схема
`Wonder` стабильна), либо на каждую новую колонку `Wonder` пишем `ALTER TABLE
wonders ADD COLUMN IF NOT EXISTS ...` в `base.py` (как для остальных таблиц).

## Ассеты (карта)
Пайплайн карты: тайлы `/world/tiles/...webp` + PNG-спрайты маркерами (как таверны
`map_tavern_N.png` через `_atlas_pos`). Чуду нужен спрайт ~128–220px, вид
спереди/сбоку (не изометрия), 3 стадии ИЛИ 1 готовый + оверлей «леса» (стадии
фейкаются CSS-clip, как анимации орков/героев). Источники: itch.io
`tag-castle`/`tag-city-builder`+`tag-pixel-art`, Kenney Castle Kit (CC0), CraftPix.
Строим на плейсхолдере — арт вставляем в Фазе 3, не блокируя код.
