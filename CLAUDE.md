Нужно разработать Telegram-бота с текстовой многопользовательской игрой на кнопках.

Игра представляет собой симулятор средневековой таверны/паба. Каждый игрок из общего Telegram-чата может создать свою таверну, развивать её, строить пристройки, собирать ресурсы, крафтить товары, взаимодействовать с другими игроками и участвовать в событиях общего мира.

Бот должен работать в двух режимах:

1. Личные сообщения с ботом — основной игровой интерфейс.
2. Общий Telegram-чат — публичные события, новости мира, рейтинги и взаимодействие сообщества.

Проект должен быть построен так, чтобы сначала реализовать MVP, а затем постепенно расширять игру новыми механиками.

Формат игры:

- текстовая Telegram-игра;
- управление через inline-кнопки;
- статичные изображения;
- асинхронная серверная логика;
- многопользовательская экономика;
- общий мир для участников одного Telegram-чата.

Игрок не вводит сложные команды. Основное управление происходит через кнопки Telegram.

Игрок начинает с маленькой таверны и постепенно развивает её.

Основной цикл:

1. Игрок запускает бота.
2. Создаёт свою таверну.
3. Получает стартовые ресурсы.
4. Собирает ресурсы по кулдауну.
5. Строит новые здания и пристройки.
6. Улучшает таверну.
7. Крафтит товары.
8. Получает доход и репутацию.
9. Взаимодействует с другими игроками.
10. Участвует в событиях общего мира.

Игрок соответствует Telegram-пользователю.

У игрока должны быть:

- Telegram ID;
- username;
- имя;
- дата регистрации;
- название таверны;
- уровень;
- золото;
- репутация;
- регион на карте;
- статус активности.

У каждого игрока есть одна таверна.

Параметры таверны:

- название;
- уровень;
- вместимость;
- комфорт;
- доход;
- репутация;
- текущие улучшения;
- построенные пристройки.

Таверна должна отображаться отдельным экраном с текстовым описанием, параметрами и кнопками управления.
